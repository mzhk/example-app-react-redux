# example-app-react-redux
Minimal example app - react, redux, babel 7, webpack 4

## usage
```bash
git clone https://github.com/mzhk/example-app-react-redux.git
cd example-app-react-redux
npm install
```

## dev mode
```bash
npm run start
open http://localhost:8080/
```

## build
```bash
npm run build
```
